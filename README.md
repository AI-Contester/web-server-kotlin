# web-server-kotlin
It is implementation of AI-Contester web server in Kotlin using Spring

## Build
``` shell
gradle build
```

## Run
``` shell
cd build/libs
java -jar web-server-0.1.0.jar
```

## Configuration
### Port
``` shell
--port=1234
```
#### Example
``` shell
java -jar web-server-0.1.0.jar --port=1234
```

## Development run
``` shell
gradle bootRun
```
